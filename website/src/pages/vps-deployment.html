Prerequisites

Before you begin, make sure you have:

  • A SolarBot subscription (Basic, Advanced, or Professional)
  • A VPS with at least the following specifications:
    • 1 CPU core
    • 2GB RAM
    • 20GB SSD storage
    • Ubuntu 20.04 LTS or later
  • SSH access to your VPS
  • A Solana wallet with SOL for transaction fees
  • USDC or WSOL tokens for trading
  • A Telegram bot token (obtained from @BotFather)

Automatic Deployment (Recommended)

We provide an automatic deployment script that handles all the necessary setup steps:

Step 1: Connect to Your VPS

ssh username@your-vps-ip

Step 2: Download the Deployment Script

wget https://solarbot.io/deploy-vps.sh

Step 3: Make the Script Executable

chmod +x deploy-vps.sh

Step 4: Run the Script

sudo ./deploy-vps.sh

The script will guide you through the setup process and prompt you for the necessary information.

Note

Advanced and Professional tier subscribers receive personalized setup assistance. Contact our support team if you need help with deployment.

Manual Deployment

If you prefer to set up the bot manually, follow these steps:

Step 1: Update System Packages

sudo apt update
sudo apt upgrade -y

Step 2: Install Dependencies

sudo apt install -y curl git build-essential

Step 3: Install Node.js

curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

Step 4: Create a User for the Bot

sudo adduser --disabled-password --gecos "" solarbot

Step 5: Clone the Repository

cd /home/solarbot
sudo git clone https://github.com/arodrig125/solana-trading-bot.git
sudo chown -R solarbot:solarbot solana-trading-bot

Step 6: Install Bot Dependencies

sudo -u solarbot bash -c "cd /home/solarbot/solana-trading-bot && npm install"

Step 7: Install PM2 for Process Management

sudo npm install -g pm2

Configuration

After deployment, you need to configure the bot with your credentials:

Step 1: Create Environment File

sudo -u solarbot bash -c "cd /home/solarbot/solana-trading-bot && cp .env.example .env"

Step 2: Edit Environment File

sudo nano /home/solarbot/solana-trading-bot/.env

Add the following information to the .env file:

# Telegram Configuration
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_telegram_chat_id

# Solana Wallet Configuration
PRIVATE_KEY=[your_wallet_private_key_array]

# Bot Operation Mode
SIMULATION=true  # Set to false for live trading (Advanced and Professional plans only)

# RPC Endpoint (Optional)
RPC_ENDPOINT=https://api.mainnet-beta.solana.com

Warning

Never share your private key or .env file with anyone. Keep your credentials secure at all times.

Step 3: Start the Bot

sudo -u solarbot bash -c "cd /home/solarbot/solana-trading-bot && pm2 start server.js --name solarbot"

Step 4: Configure PM2 to Start on Boot

sudo env PATH=$PATH:/usr/bin pm2 startup systemd -u solarbot --hp /home/solarbot
sudo -u solarbot bash -c "pm2 save"

Telegram Setup

SolarBot uses Telegram for command control and notifications. Follow these steps to set up your Telegram bot:

Step 1: Create a Telegram Bot

  1. Open Telegram and search for @BotFather
  2. Start a chat with BotFather and send the command /newbot
  3. Follow the prompts to name your bot and create a username for it
  4. Once created, BotFather will provide you with a token
  5. Copy this token and add it to your .env file as TELEGRAM_BOT_TOKEN

Step 2: Get Your Chat ID

  1. Start a chat with your newly created bot
  2. Send any message to the bot
  3. Start the bot with the steps above
  4. Send the command /chatid to your bot
  5. The bot will respond with your Chat ID
  6. Copy this ID and add it to your .env file as TELEGRAM_CHAT_ID

Step 3: Restart the Bot

sudo -u solarbot bash -c "pm2 restart solarbot"

Monitoring

You can monitor the bot using PM2:

View Logs

sudo -u solarbot bash -c "pm2 logs solarbot"

Monitor Process

sudo -u solarbot bash -c "pm2 monit"

Restart the Bot

sudo -u solarbot bash -c "pm2 restart solarbot"

Stop the Bot

sudo -u solarbot bash -c "pm2 stop solarbot"

Troubleshooting

Bot Not Starting

Check the logs for errors:

sudo -u solarbot bash -c "pm2 logs solarbot"

Connection Issues

If the bot is having trouble connecting to the Solana network, try using a different RPC endpoint in your .env file.

Telegram Bot Not Responding

  1. Make sure your TELEGRAM_BOT_TOKEN is correct
  2. Ensure your bot is not blocked or stopped
  3. Check that your TELEGRAM_CHAT_ID is correct

Out of Memory Errors

If you're experiencing out of memory errors, consider upgrading your VPS to a higher tier with more RAM.
